🚌🎫 Online Bus Reservation System

The Online Bus Reservation System is a desktop-based application developed using Java and MySQL that simplifies the process of booking bus tickets.
The system provides an easy and user-friendly platform for passengers to register, view bus details, book tickets, and make payments digitally without standing in long queues.
The project was developed as a part of the TY BSc Computer Science curriculum.
-------------------------------------------------------------------------------------------------------------
📌 Features

 User Module
- User Registration and Login
- View Bus Routes and Timings
- Book Bus Tickets
- Ticket Confirmation
- QR Code Payment
- View Ticket Details

 Admin Module
- Admin Login
- Add Bus Details
- Add Route Details
- Update Bus Information
- Remove Bus Information
- Manage Route and Ticket Data

📊 Project Modules

 1. Login and Registration
Allows users and admin to securely log into the system. New users can create accounts using the registration form.

 2. Bus Management
Admin can add, update, and remove bus details along with timings and routes.

 3. Route Management
Routes including source, destination, and fare details are managed through the admin panel.

 4. Ticket Booking
Users can select buses, enter passenger details, and confirm bookings.

 5. Payment System
QR-based payment interface is provided for ticket payment.
